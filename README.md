
# WeChatDemo
## About
基于XMPP/Openfire的即时聊天Demo  
实现用户的登陆注册注销并保存到服务器  
即时聊天、发送语音、图片、文件

## Usage
### Download

		git clone https://github.com/w11p3333/WechatDemo.git
### install
配置openfire服务器
		Product>CocoaPods>installpods
